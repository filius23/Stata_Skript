So können wir einen Datensatz in Stata einlesen:

use "D:\Studium\01_Oldenburg\Lehre\Datensaetze\Allbus 1980-2016\ZA4586_v1-0-0.dta" , clear

Mit keep können wir bestimmte Zeilen auswählen, inrange(variable,von,bis) ist eine hilfreiche Abkürzung dafür:

keep if year == 2010 & educ > 0 & inrange(educ,1,5)
(61,894 observations deleted)
keep if inc > 0 & work == 1
(1,732 observations deleted)
keep if age > 0
(1 observation deleted)
